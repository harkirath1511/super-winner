# super-winner

Simple Next.js + TypeScript project with a minimal Supabase setup.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create your env file:

   ```bash
   cp .env.example .env.local
   ```

3. Fill in your Supabase values in `.env.local`.

4. Start the app:

   ```bash
   npm run dev
   ```

## Available scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
